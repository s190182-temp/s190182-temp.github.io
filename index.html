What is Python?

Python is a high-level, general-purpose, interpreted language. It emphasizes on code readability and clarity, while providing useful features that abides by its 'Pythonic' philosophy. It is a great language for starters due to it's simple syntax; for scripting due to it's brevity and out-of-the-box features; for scientific computing due to its comprehensive libraries like Numpy, Pandas, and TensorFlow.

Features

data structures

Lists

my_list = [1, 2, 3, 4]
my_list.append(5)               # append 5 to the end of list
my_list.pop(1)                  # remove value at index 1 and returns it
my_list2 = my_list[1:3:-1]      # list slicing, returns my_list from index 1(inclusive) to index 4(exclusive) with step -1: [4,3,1]

Dictionaries

my_dict = {"name": "John", "id": 893254383, "age", 30}   # dict keys must be immutable(i.e. no list/set/dict...)
id_num = my_dict["id"]      # returns 893254383
my_dict["age"] += 1         # add value of "age" by 1
all_items = my_dict.items() # returns an iterable containing (key, value) tuples

Sets

my_set = set()               # use set() instead of {} to create an empty set as {} create a dict instead

my_set = set([10,5,7,32,7,2,12,8,2])    # my_set -> {32,2,5,7,8,10,12}, as set is unordered and removes duplicates.

n = {1,3,5} | {2,4,6}           # set union, returns {1,2,3,4,5,6}, both combined
n = {2,4,6} & {3,6,9}           # set intersection, returns {6}, common elements
n = {1,2,3,4,5} - {1,3,5}       # set difference, returns {2,4}, elements only in the first set
n = {2,4,6} ^ {3,6,9}           # set symmetric difference, returns {2,3,4,9}, elements in either set but not both

# Note that numbers in set may sometimes appear sorted,
# but that is due to set's use of hash() and isn't always the case.


f-strings

formatted string literals, a more convenient way to format strings and process expressions.

age = 4
name = "John"
intro = f"{name} is {age} years old."   # returns "John is 4 years old."

n = math.sin(30)
rounded = f"{n:.3f}"                    # returns "-0.988", with n corrected to 3 decimal place

n = 7
n_in_bin = f"{n:b}"                     # returns "111", with 7 converted to binary


List comprehension, map(), and filter()

compact and simple way to create lists as an alternative to for-loops.

my_list = []
for i in range(10):
    if i % 2:
        my_list.append(i)
my_list                                         # [1,3,5,7,9]

my_list = list(lambda i : i % 2, range(10)))    # also [1,3,5,7,9]
my_list = [i for i in range(10) if i % 2]       # also [1,3,5,7,9]


strs = ["1","2","3","4","5"]
numbers = []
for i in strs:
    numbers.append(int(i))
numbers                                         # [1,2,3,4,5]

numbers = list(map(int, strs))                  # also [1,2,3,4,5]
numbers = [int(i) for i in strs]                # also [1,2,3,4,5]
